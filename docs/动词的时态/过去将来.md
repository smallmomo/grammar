# 一、四种过去将来时态完整结构整理
主语分两类：
1. 第一人称 I / we → 可用 should / would
2. 二、三人称 you/he/she/it/they → 只用 would

## ## 1. 一般过去将来时：would / should + 动词原形
结构：would work / should work
含义：站在过去视角，描述**当时将要发生的动作、计划**
例句：
He said he would work hard.
他说他会努力工作。

## ## 2. 过去将来进行时：would / should + be doing
结构：would be working / should be working
含义：站在过去看将来，将来某个时刻**正在做某事**
例句：
She told me she would be working at 8 p.m.
她跟我说晚上八点她会正在工作。

## ## 3. 过去将来完成时：would / should + have done
结构：would have worked / should have worked
含义：站在过去视角，将来某时间前**已经做完**
例句：
I thought I would have finished the work by noon.
我当时以为中午前就能做完工作。

## ## 4. 过去将来完成进行时：would / should + have been doing
结构：would have been working / should have been working
含义：站在过去预判，到未来某个时间点，动作**持续很久**
例句：
He said by next year he would have been working here for 10 years.
他说等到明年，他就已经在这连续工作十年了。

# 二、极简区分口诀
### 一般过去将来 would do
过去视角，单纯“将要做”
### 过去将来进行 would be doing
过去视角，将来某刻“正在做”
### 过去将来完成 would have done
过去视角，将来节点前“已做完”
### 过去将来完成进行 would have been doing
过去视角，到将来节点“持续做了很久”

# 三、补充小知识点
现代英语里，不管第几人称，几乎统一只用 would；should 多用于老旧书面、虚拟语气，日常口语很少用 should 表过去将来。
